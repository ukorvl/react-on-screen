## Getting started
Using npm
```bash
$ npm i @nilfoundation/react-components
```
or yarn
```bash
yarn add @nilfoundation/react-components
```
or CDN
```html
<script src="https://unpkg.com/@ukorvl/react-on-screen"></script>
```
