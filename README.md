## Getting started
Using npm
```bash
$ npm i @ukorvl/react-on-screen
```
or yarn
```bash
yarn add @ukorvl/react-on-screen
```
or CDN
```html
<script src="https://unpkg.com/@ukorvl/react-on-screen"></script>
```
